
# Current Games

Steep steps, Ski race, Universal script, Walk to school in Ohio, Roblox chaos mod.




## How to use key saving feature

This is only working for steep steps script **ONLY**

```lua
-- leave empty if you dont have any of the keys

_G.KeyGui = false

_G.FreeKey = ""
_G.NoKey = false
_G.BoostKey = "" 
_G.BoughtKey = ""

```
